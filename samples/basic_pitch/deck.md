# Markdown Present Pitch

---

## The Problem
Running presentations effectively shouldn't require complex software.
Sharing out the current slide in real-time shouldn't require sending bulky PDFs.

---

## The Solution
**Markdown Present**
A tool designed to:
- Be fast & lightweight
- Allow presenter controls
- Synced views for the audience

---

## Tech Stack
- Frontend: Vanilla HTML/JS + Reveal.js
- Backend: Python FastAPI + WebSockets
- Storage: Ephemeral Server Storage & LocalForage (In-Browser)

---

## Why use it?
Clean, frictionless sharing. 

> "It just works, out of the box."
